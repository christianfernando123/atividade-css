# Atividade CSS

## 📌 O que é CSS

CSS (Cascading Style Sheets) é usado para estilizar páginas HTML.

## 🎨 Tipos de CSS

### CSS Inline

Aplicado diretamente na tag HTML.

### CSS Interno

Definido dentro da tag `<style>` no HTML.

### CSS Externo

Arquivo separado `.css` conectado ao HTML.

---

## 🔎 Seletores CSS

Seletores são usados para escolher quais elementos serão estilizados.

### Tipos de seletores:

1. **Seletor de elemento**
2. **Seletor de classe**
3. **Seletor de ID**
4. **Seletor universal**
5. **Seletor descendente**
